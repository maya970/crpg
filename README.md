## ✨ 游戏特色

- **全链上地牢**：Move 合约驱动的核心逻辑（`adventurer::dungeon`），无后端服务器，防作弊、可验证
- **沉浸式 3D 渲染**：Three.js 实时渲染地城场景，丝滑的 Web 体验
- **一键链上交互**：通过 InterwovenKit 钱包连接，即可战斗、探索、获得 NFT 装备
- **测试网已上线**：随时可玩，无需等待主网
- **开源透明**：合约 + 前端全部开源，欢迎社区共建

## 🛠️ 技术栈

- **区块链**：Initia（Move 语言）
- **智能合约**：`move/` 目录下的 `adventurer::dungeon` 模块
- **前端**：React + Vite + Three.js + InterwovenKit
- **钱包集成**：InterwovenKit（支持自动签名、Privy）
- **部署**：Vercel 静态托管（一键部署）
